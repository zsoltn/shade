Example
=======

Sometimes an example is nice.

#. Lists to the output::

     python inventory.py --list 

#. Lists to Excel (./inventory.xls)::

     python inventory.py --list --excel
     
#. Lists to a Tree::

     python inventory.py --list --tree

     
#. Lists to Graph (./inventory.html)::

     python inventory.py --list --graph
